# Flowbar - Product Requirements Document

## 제품 개요

**Flowbar**는 macOS 메뉴바를 상황별로 자동 전환하고, 방해를 차단하여 집중력을 높이는 유틸리티 앱입니다.

### 문제 정의

1. **메뉴바 혼란**: 15개 이상의 메뉴바 아이콘이 공간을 차지, 노치 디스플레이에서 더 심각
2. **Focus stealing**: 작업 중 팝업이 키보드 입력을 가로채는 문제
3. **Launchpad 제거**: macOS 26에서 Launchpad가 사라짐
4. **컨텍스트 스위칭 비용**: 작업 전환 시 매번 설정을 다시 조정

### 해결책

상황별로 적절한 메뉴바 구성을 보여주고, 집중을 방해하는 요소를 차단합니다.

## 핵심 기능

### MVP (v1.0)

#### 1. 스마트 메뉴바 그룹핑

```
사용자가 "모드"를 정의
→ 각 모드마다 표시할 메뉴바 아이콘 지정
→ 단축키 또는 메뉴바 클릭으로 전환
```

**기본 모드 예시:**
- `코딩` → Git, CI 타이머, CPU
- `디자인` → 컬러 피커, 에셋 폴더
- `회의` → 캘린더, 노트, 마이크
- `집중` → 타이머만, 나머지 숨김

#### 2. Focus 가드 (기본)

- 새 창이 열릴 때 키보드 포커스를 가로채지 않음
- "방해 금지 앱" 리스트 설정
- 알림 배너 자동 숨김 (선택)

#### 3. 미니 런처

- Launchpad 대체 (간단한 버전)
- 현재 모드에 맞는 앱만 표시
- 키워드 검색

### v1.5

- 워크플로우 저장: 앱 + 윈도우 위치 + 메뉴바 구성
- 스케줄 기반 모드 전환 (예: 9시에 "업무", 18시에 "개인")
- Apple Shortcuts 연동

### v2.0

- AI 기반 모드 추천
- "지금 이 앱 켜면 좋을까?" 제안
- 팀 워크플로우 공유

## 사용자 플로우

### 온보딩

```
1. 설치 후 실행
2. 기본 모드 4개 자동 생성 (코딩, 디자인, 회의, 집중)
3. 현재 메뉴바 아이콘 스캔
4. 각 모드에 아이콘 할당 (드래그 앤 드롭)
5. 완료
```

### 일반 사용

```
1. 메뉴바에 Flowbar 아이콘 표시
2. 클릭 → 모드 선택
3. 선택한 모드의 아이콘만 표시
4. Focus 가드 자동 활성화
```

### 단축키

- `⌘ + Shift + M` → 모드 전환 팝업
- `⌘ + Shift + 1~9` → 모드 1~9로 바로 전환

## 기술 스펙

### 권장 스택

| 항목 | 기술 | 이유 |
|-----|------|------|
| 언어 | Swift 5.9 | 최신 macOS 기능 활용 |
| UI | SwiftUI + AppKit 하이브리드 | 빠른 개발 + 강력한 기능 |
| 데이터 | SwiftData | 로컬 저장 |
| 메뉴바 조작 | NSStatusItem, Accessibility API | 시스템 레벨 제어 |
| Focus 제어 | NSWorkspace, Apple Events | 윈도우 이벤트 감지 |

### 시스템 요구사항

- macOS 13.0 Ventura 이상
- Apple Silicon + Intel 지원
- 크기: < 10MB
- 메모리: < 50MB

### 권한

| 권한 | 용도 | 필수 여부 |
|-----|------|----------|
| Accessibility | 메뉴바 아이콘 조작 | 필수 |
| Automation | 앱 제어 | 선택 |
| Notifications | 알림 관리 | 선택 |
