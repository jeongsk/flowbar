# Flowbar Privacy Policy

**Last Updated:** February 16, 2026

## Introduction

Flowbar ("we," "our," or "us") respects your privacy and is committed to protecting your personal data. This privacy policy explains how we collect, use, and safeguard your information when you use Flowbar.

## Information We Collect

### 1. Automatically Collected Information

Flowbar may collect the following information automatically:

- **System Information**: macOS version, device model, and hardware specifications
- **Usage Data**: Which modes you use most frequently, which apps you launch, and how often you switch modes
- **Performance Data**: Memory usage, CPU usage, and application performance metrics
- **Crash Reports**: Information about application crashes and errors

### 2. Voluntarily Provided Information

When you submit feedback or bug reports, we may collect:
- **Feedback Content**: Your descriptions, suggestions, and bug reports
- **Contact Information**: Only if you voluntarily provide it in GitHub issues or discussions

### 3. Accessibility Data

Flowbar requires Accessibility permission to:
- Detect menu bar icons
- Control icon visibility
- Prevent focus theft
- Monitor application focus changes

**This data is processed locally on your device and is never transmitted to our servers.**

## How We Use Your Information

We use the collected information to:

1. **Improve Flowbar**: Analyze usage patterns to enhance features and performance
2. **Fix Bugs**: Use crash reports and performance data to identify and resolve issues
3. **Provide Support**: Respond to your feedback and support requests
4. **Develop New Features**: Understand user needs to prioritize feature development

## Data Storage and Processing

### Local Storage

All personal data is stored locally on your device:
- **SwiftData Database**: Your modes, icon assignments, preferences, and app assignments
- **Preferences**: User settings and customization options
- **Backup Files**: Encrypted backups stored in your Application Support directory

### Cloud Storage

Flowbar does not use cloud storage for personal data. All data remains on your device unless you explicitly:
- Submit a GitHub issue (public)
- Send an email (private)
- Participate in discussions (public)

## Data Sharing

We do **not** sell, rent, or share your personal data with third parties for marketing purposes.

### GitHub Issues

When you submit bug reports or feature requests via GitHub:
- **Public**: Your submission is visible to the public
- **Data**: Only what you explicitly include in your submission
- **Control**: You choose what information to share

### Analytics

Flowbar may collect anonymous usage analytics (opt-in only):
- **Data**: Aggregated statistics about feature usage
- **Personal Information**: None
- **Control**: You can disable analytics in Settings

## Data Security

We implement reasonable security measures to protect your information:

- **Local Encryption**: Backup files are encrypted using device-specific keys
- **Access Control**: Only Flowbar can access its database
- **Secure Transmission**: Any network communication uses HTTPS/TLS

However, no method of transmission is 100% secure. While we strive to protect your data, we cannot guarantee absolute security.

## Third-Party Services

Flowbar integrates with the following third-party services:

### GitHub
- **Purpose**: Issue tracking, discussions, and source code hosting
- **Data**: Only what you voluntarily provide in issues or discussions
- **Policy**: Subject to GitHub's Privacy Policy

### macOS System APIs
- **Accessibility API**: For menu bar icon control
- **CGEvent Tap**: For focus protection
- **NSWorkspace**: For app launching and management
- **Policy**: These APIs operate within Apple's security framework

## Data Retention

### Local Data
- **Duration**: Retained until you uninstall Flowbar or reset settings
- **Backups**: Last 5 backup files maintained automatically
- **Control**: You can delete data at any time through Settings

### Public Data
- **GitHub Issues**: Retained indefinitely (public record)
- **Control**: You can edit or close your GitHub issues

## Your Privacy Rights

You have the right to:

1. **Access**: View your data through Settings or by accessing the SwiftData database
2. **Delete**: Remove all Flowbar data by:
   - Using "Reset All Settings" in Settings
   - Deleting Flowbar's application support directory
   - Uninstalling Flowbar
3. **Opt-out**: Disable analytics in Settings
4. **Export**: Export your data using the backup feature

## Children's Privacy

Flowbar is not intended for children under 13. We do not knowingly collect information from children under 13.

## Changes to This Policy

We may update this privacy policy from time to time. We will notify you of significant changes by:
- Posting the new policy on GitHub
- Updating the in-app privacy notice
- Sending a notification (if you have notifications enabled)

## Accessibility Permission Details

Flowbar requires Accessibility permission to function properly. Here's what we do with this permission:

### What We Do:
- Scan menu bar to detect icons and their positions
- Hide/show menu bar icons based on current mode
- Monitor application focus changes
- Prevent other apps from stealing focus

### What We Don't Do:
- Read the content of your files or emails
- Capture keystrokes or passwords
- Access your camera or microphone
- Read notifications or messages
- Access your internet history or downloads

### Technical Details:
- All accessibility operations happen locally on your device
- No accessibility data is transmitted over the network
- Accessibility permission can be revoked at any time in System Settings
